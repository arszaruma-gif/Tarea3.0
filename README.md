## 🚀 Descripción del Proyecto

Este proyecto consiste en una aplicación básica en Python que incluye pruebas unitarias automatizadas mediante `pytest`. Además, cuenta con un flujo de Integración y Entrega Continua (CI/CD) configurado con **GitHub Actions**, el cual compila, prueba, simula un despliegue y publica una imagen Docker en **GitHub Container Registry (GHCR)**.
